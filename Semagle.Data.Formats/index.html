Semagle.Data.Formats Library

Semagle.Data.Formats provides reading and writing functions for popular machine learning dataset formats.

LIBSVM

LIBSVM uses "sparse" vector format where zero values do not need to be stored. Each vector is repsented by a separate text line. A first number of the line is the label y (class number or value of the approximated function), which is followed by index/value pairs of non-zero elements of the feature vector x.
