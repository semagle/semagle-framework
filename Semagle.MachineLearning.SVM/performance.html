Semagle.Framework :: Semagle.MachineLearning.SVM

Performance

Speed is one of the main concerns of machine learning algorithms developers. The native SVM implementation (LIBSVM 3.20) is optimized for specific kernel types and demonstrates good training and test times for small and large problems. The F# implementation running on Mono 5.10 is 5-6 times slower than the native SVM that makes it unsuitable for large problems. However, with parallel kernel evaluations the F# implementation is only two times slower that makes it practically useful.
